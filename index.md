Introduction to C Programming
------

This *Introduction to C Programming* (*introC*) course is part of the ACE program hosted by the [Information Networking Institute](https://www.cmu.edu/ini) at [Carnegie Mellon University](https://www.cmu.edu/).

## Course Description

The C programming language has stood as one of the most popular programming languages for multiple decades and is the core of our world's infrastructure and computing systems. As a low-level programming language, C gives a system programmer direct access to hardware resources, making it well-suited for use in kernel and operating system design, embedded and robotics applications, and critical infrastructure, just to name a few.  Based on these motivations, the goal of this course is to build and strengthen low-level programming skills and ensure foundational abilities for future work in systems.

## Learning Objectives

In taking and completing this course, students will:
* Build foundational background in C programming
* Understand foundational concepts, abilities, risks, and shortcomings of the C programming language and its uses in system and application programming
* Practice and gain hands-on experience writing and manually debugging C programs

## Modules

The course contains eight different modules of varying difficulty, each comprising several sub-modules.  A single sub-module includes a recorded lecture video, a brief quiz, and a programming task, which combine to provide background knowledge and training as well as hands-on experience using the C programming language.  The eight modules included in the course are:
* Getting Started
* Writing and Running C Programs
* Variables and Operators
* Program Flow Control
* More Variables and Operators
* Memory and Pointers
* Libraries
* Additional Important Features

## Programming Infrastructure

The *introC* course uses GitHub to initialize and maintain students' code repositories for the various programming tasks that are a required component of the course.  In addition, there is an external auto-grading service that interacts with student repositories using the GitHub API.
